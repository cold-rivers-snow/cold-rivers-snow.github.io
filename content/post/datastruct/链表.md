双向链表，删除 prev 则为单链表。

```cpp
template <typename E>
class Node {
public:
    E val;
    Node* next;
    Node* prev;
    Node(Node* prev, E element, Node* next) {
        this->val = element;
        this->next = next;
        this->prev = prev;
    }
};
```

**链表的元素可以分散在内存空间的天涯海角，通过每个节点上的 next, prev 指针，将零散的内存块串联起来形成一个链式结构。**

好处：

1. 提高内存利用率；只要 new 出来就能用
2. 没有容量限制；用的时候就连接上，不用就断开

局限性：**数组最大的优势是支持通过索引快速访问元素，链表没有**

1. 同时持有头尾节点的引用
   1. 提高尾部插入的时间复杂度
2. 虚拟头尾节点
   1. 虚拟节点是你内部实现数据结构的技巧，对外是不可见的
3. 内存泄漏
   1. delete 指针 ，并置为空
